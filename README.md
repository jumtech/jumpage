## what's this?
- [jumpei_ikegami](https://twitter.com/jumpei_ikegami)のポートフォリオページです
- ソースコードと実装方法を公開しているので、学習用途でも使えます

## 技術スタック
- node.js:7.7.2
- Express:4.15.2
- Docker

## setup

```
cd app
npm install
sh ../scripts.docker_run.sh
```
